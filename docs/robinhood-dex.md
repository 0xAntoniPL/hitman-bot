# Robinhood DEX support limitations

Supported in this fork:
- Uniswap V2 style router (`RH_UNI`) on mainnet chain id `4663`
- Manual contract-address swaps (ETH -> token)

Not supported yet:
- Universal Router
- Uniswap V3 / V4 routing
- Cross-DEX aggregators
- Full Telegram scanner sniping flows on Robinhood

Always re-verify router and token addresses on Blockscout before using real funds.
